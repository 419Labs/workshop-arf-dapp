**Modern**, **Flexible** Starknet Dapp Template.


## Developing

Clone the repository

```bash
git clone git@github.com:a5f9t4/cairopal.git
cd cairopal
```

Install Dependencies

```bash
yarn
```

Run locally

```bash
yarn dev
```

#### Run tests

```bash
yarn test
```
